## REPL

- Here we can test small pieces of code instantly without making a file and using compiler .

- All Intrepreted language always have REPL.

- REPL is usefull because i t allows you to :
  - test code quickly
  - write some code quickly
  - To learn and practice

- I also learned that in REPL the four steps happen in repeat :

- Read -Reads the code
- Evaluate - Evaluate/Execute the code
- Print - Dispaly the result
- Loop -Waits for the next command

#### Example

- open the terminal type `node` then

> console.log("Hello World");

> Hello World

> Undefined

- we can exit the REPL using `.exit` command

## CLI Argument

- Command Line Arguments are the input/data that we give with the commands .

eg: mkdir js-practice

- Flags are the special commands we give the commands while working with the CLI

- they looks like -v , --oneline,--help ...

Eg: node --version
