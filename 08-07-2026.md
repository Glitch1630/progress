### **Commands in Git Bash and Powershell**

I learned basic commands equivalent in both Git Bash and Powershell and they are :

### Git Bash

- `pwd` – Shows the current working directory.
- `ls` – Lists files and folders.
- `cd` – Changes the current directory.
- `mkdir` – Creates a new directory.
- `touch` – Creates a new file.
- `cat` – Displays the contents of a file.
- `cp` – Copies a file.
- `mv` – Moves or renames a file.
- `rm` – Deletes a file.

### PowerShell

- `pwd` – Shows the current working directory.
- `ls` / `dir` – Lists files and folders.
- `cd` – Changes the current directory.
- `mkdir` – Creates a new directory.
- `New-Item` – Creates a new file.
- `Get-Content` (or `cat`) – Displays the contents of a file.
- `Copy-Item` – Copies a file.
- `Move-Item` – Moves or renames a file.
- `Remove-Item` (or `del`) – Deletes a file.

I learned commands to create folders, create files , removing files , moving files between directories ,clearing the console , copying a file form one place to another , renaming the folder.
